h1. Chart Libraries

h2. Kostenfreie Libraries

p. Die Bloonix-WebGUI verwendet zur visuellen Darstellung von Statistiken diverse JavaScript-Libraries. Für die kostenfreie Variante werden die Libraries Flot und JQVMap eingesetzt. Diese Libraries stehen unter der MIT Lizenz und erlauben somit eine kommerzielle Nutzung. Weitere Details zu den Libraries gibt es hier: "LICENSE-WEBGUI":https://download.bloonix.de/LICENSE-WEBGUI

h2. Lizenzen für Highcharts und Highmaps

p. Die Bloonix-WebGUI ist zur visuellen Darstellung für den Einsatz von Highcharts und Highmaps optimiert. Entgegen der Lizenzen von Bloonix darf die Software von Highcharts und Highmaps jedoch nicht für kommerzielle Zwecke verwendet werden. Falls Sie dennoch den Wunsch haben, Highcharts und Highmaps anstelle der kostenfreien Tools Flot und JVQMap einzusetzen, so müssen Sie beim Anbieter Highsoft eine Lizenz erwerben.

p. Weitere Details zu Highcharts und Highmaps gibt es auf den Webseiten "http://www.highcharts.com":http://www.highcharts.com/ und "http://shop.highsoft.com/":http://shop.highsoft.com/ .

p. *Hinweis:* Die Bloonix-WebGUI wird bereits mit der JavaScript-Library für Highcharts und Highmaps ausgeliefert. Falls Sie eine Lizenz für Highcharts und Highmaps erworben haben, so können Sie den Einsatz der Chart-Libraries über einen Konfigurationsparameter einschalten. Dieser befindet sich in der Konfigurationsdatei __/etc/bloonix/webgui/main.conf__ und lautet __chart_library__. Weitere Informationen können in der Konfigurationsdatei nachgelesen werden.
